# 실습 #1 : TDD를 이용한 사칙 연산 계산기 구현

## 목표

- TDD를 통한 코드 작성을 경험한다.
- 작성된 테스트 코드 보호 아래 Interface와 전략 패턴을 활용하여 DIP와 추상화를 만족하는 확장 가능한 코드로의 리팩토링을 경험한다.

## 요구 사항

- [ ] Public으로 작성된 `모든 메서드에 테스트 코드를 작성`한다.
- [ ] `두개의 피 연산자와 하나의 연산자`를 입력 받아 `연산하고 결과를 반환`한다.
- [ ] 피 연산자는 `양수만 허용`한다.
- [ ] 연산자는 `+, -, *, /만 허용`한다.
- [ ] `0으로 나누는 경우` 예외를 발생시킨다.
- [ ] 확장을 고려하여 설계한다.
    - 원시 타입 포장 객체 적용
    - 전략 패턴 적용
